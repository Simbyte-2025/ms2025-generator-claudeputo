# Documentación Técnica - MS2025 Site Generator

Este directorio contiene la documentación técnica y evidencias del proyecto.

## Estructura

- **BUILD_MANIFEST.md** - Manifiesto de build con versiones exactas y comandos
- **DEPLOY_PLAN.md** - Plan de deployment (local → staging → producción)
- **TOKENS_BUDGET.md** - Estrategia de minimización de tokens para Manus
- **AUDIT_REPORT.md** - Reporte de auditoría técnica con hallazgos
- **artifacts/** - Artefactos de build (ZIPs, checksums)
- **lighthouse/** - Reportes de Lighthouse en formato JSON

## Propósito

Esta documentación está diseñada para:
1. Garantizar reproducibilidad del build
2. Facilitar deployment automatizado vía Manus
3. Minimizar consumo de tokens en operaciones de Manus
4. Proporcionar evidencias de calidad y seguridad
